# Form Handling in React

This project demonstrates two approaches to form handling in React:

1. Controlled components with useState and manual validation
2. Formik with Yup for schema-based validation

## Features
- User registration form (username, email, password)
- Real-time input handling
- Basic error display
- Form submission logging

## Setup
npm install
npm run dev

## Technologies
- React + Vite
- Formik & Yup
